# R Random Project

A simple R project for exploration and experiments.

## Structure

- `main.R` - Main entry point script
- `utils.R` - Utility/helper functions
- `analysis.R` - Data analysis scripts

## Getting Started

1. Open RStudio or your preferred R environment
2. Set working directory to this folder
3. Run `source("main.R")` to get started

## Requirements

- R (>= 4.0.0 recommended)
- Optional packages: `tidyverse`, `ggplot2`
