**Résumé — Rev comme capture canonique de l’arrêt**

Nous modélisons une exécution par une *trace* `T : ℕ → Prop` et définissons l’arrêt par `Halts T := ∃ n, T n`. Nous introduisons un opérateur de clôture `up : Trace → Trace` défini par `up T n := ∃ k ≤ n, T k`, qui monotonise toute trace et produit des représentants canoniques *clos* (points fixes de `up`). Un *Kit* `K` est un observateur abstrait `K.Proj : Trace → Prop`, dont on n’exige la correction qu’au niveau des traces closes, via l’accord avec la vérité existentielle sur cette sous-classe. La construction centrale est l’opérateur inverse `Rev(K, T) := K.Proj (up T)`, autrement dit l’*observation après clôture*, qui force le verdict à ne dépendre que de la classe de clôture de `T`. On obtient ainsi une sémantique invariante par quotient : deux traces ayant la même clôture reçoivent le même verdict, et le non-arrêt se contracte en une classe négative canonique (la classe du bas). Enfin, nous établissons un théorème de canonicité : tout Kit satisfaisant la spécification sur les traces closes induit une extension sur toutes les traces extensionnellement équivalente à l’arrêt standard. Rev ne décide pas l’arrêt ; il le capture en imposant une géométrie d’observation canonique où l’arrêt est l’unique extension compatible.

**Abstract — Rev as a Canonical Capture of Halting**

We model executions as *traces* `T : ℕ → Prop` and define halting by `Halts T := ∃ n, T n`. We introduce a closure operator `up : Trace → Trace` given by `up T n := ∃ k ≤ n, T k`, which monotonizes any trace and yields canonical *closed* representatives (fixed points of `up`). A *Kit* `K` is an abstract observer `K.Proj : Trace → Prop`, required to agree with existential truth only on closed traces. The central construction is the reverse operator `Rev(K, T) := K.Proj (up T)`, i.e., *observation after closure*, which makes the verdict depend only on the closure class of `T`. This yields a quotient-invariant semantics: any two traces with the same closure receive the same verdict, and non-halting collapses to a unique bottom class. Finally, we prove a canonicity theorem: any Kit satisfying the closed-trace specification induces an extension on all traces that is extensionally equivalent to standard halting. Rev does not decide halting; it captures it by enforcing a canonical observation geometry in which halting is the unique compatible extension.
